### 웹 아키텍처
- 클라이언트/서버
  - 웹은 크라이언트/서버 기반 시스템으로, 클라이언트/서버 규약의 핵심 사항은 관심사의 분리다.
- 균일한 인터페이스
  - 웹을 구성하는 클라이언트. 서버, 네트워크 등의 매체 간 인터페이스는 각 인터페이스의 일관성에 기반한다.
  - 웹 컴포넌트들은 다음 네 가지의 인터페이스 제약에 따라 서로 일관성 있게 상호 운영된다.
    - 1) 리소스 식별
      - URI 같은 고유 식별자
    - 2) 표현을 통한 리소스 처리
    - 3) 자기 서술적 메시지
    - 4) 애플리케이션 상태 엔진으로서의 하이퍼미디어
